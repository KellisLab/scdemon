**scdemon** (single-cell decomposition into module networks) is a method for discovering gene co-expression modules in single-cell datasets.

Check out the :doc:`usage` section for further information, including how to
:ref:`install <installation>` the project. 

The :doc:`example` outlines a simple example for module detection on a toy dataset.

.. note::
   This project is under active development.
